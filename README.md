# Next E-COMMERCE


## Run Locally

Clone the project

```bash
  git clone https://github.com/iamNilotpal/next-ecommerce.git
```

Go to the project directories

```bash
  cd next-ecommerce/frontend
```

```bash
  cd next-ecommerce/backend
```

Install the dependencies by running

```bash
  npm install or yarn
```

Start Frontend

```bash
  npm run start
```

Start Backend

```bash
  npm run dev
```
